# react-query-rick-and-morty (AKA _Reack and Morquery_)

A React Query 'Rick and Morty' API client in TypeScript.

### Setup

Fork/clone the repo, then

```
npm i
```

### Install React Query if necessary

```
npm i react-query
```

### Running

```
npm run dev
```

### Building

```
npm run build
```
